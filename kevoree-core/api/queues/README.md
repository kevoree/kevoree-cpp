    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-lock-free-queues" class="anchor" href="#lock-free-queues" aria-hidden="true"><span class="octicon octicon-link"></span></a>Lock Free Queues</h1>

<p>Public domain implementation of four different lock free queues:</p>

<ul class="task-list">
<li>SPSC lock free dynamic queue which requires a memory allocation with each insert.</li>
<li>MPSC lock free dynamic queue which requires a memory allocation with each insert.</li>
<li>SPSC wait free bound queue/ring buffer which which uses a fixed size pre-allocated buffer.</li>
<li>MPMC lock free bound queue/ring buffer which which uses a fixed size pre-allocated buffer.</li>
</ul><h2>
<a name="user-content-performance" class="anchor" href="#performance" aria-hidden="true"><span class="octicon octicon-link"></span></a>Performance</h2>

<p>These number are on my Early 2013 Retina Macbook Pro (2.7 GHz i7). The latency variability of memory allocation can have a signifigant impact on the dynamic queues, sometimes halving throughput.</p>

<table>
